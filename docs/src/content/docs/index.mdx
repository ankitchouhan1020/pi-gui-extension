---
title: Welcome to pi-gui
description: A localhost multi-session web UI for pi.
---

pi-gui is a fast, local browser interface for the same agent, tools, models,
and sessions you already use in pi. It adds a multi-session workspace without
introducing another runtime, authentication system, or source of truth.

**pi remains the agent. pi-gui is the browser shell around it.**

---

## Start here

Install the extension from npm:

```bash
pi install npm:pi-gui-extension
```

Then run `/gui` in any pi session. pi-gui starts locally, attaches the live
session, and opens it in your browser.
